# Alert X Frontend

This is the React + Vite + TypeScript frontend for the Alert X project.

## Prerequisites

- [Node.js](https://nodejs.org/) (v18 or newer recommended)
- [npm](https://www.npmjs.com/) (comes with Node.js)

## Installation

1. **Clone the repository**  
   If you haven't already, clone the project and navigate to the frontend directory:

   ```sh
   git clone <your-repo-url>
   cd Frontend/Alert\ X\ UI/Alert\ X
   ```

2. **Install dependencies**

   ```sh
   npm install
   ```

## Development

To start the development server:

```sh
npm run dev
```

The app will be available at [http://localhost:8080](http://localhost:8080).

## Build

To build for production:

```sh
npm run build
```

## Lint

To check code style:

```sh
npm run lint
```

## Preview

To preview the production build locally:

```sh
npm run preview
```

## Configuration

- Vite config: [`vite.config.ts`](Frontend/Alert%20X%20UI/Alert%20X/vite.config.ts)
- ESLint config: [`eslint.config.js`](Frontend/Alert%20X%20UI/Alert%20X/eslint.config.js)
- Tailwind config: [`tailwind.config.ts`](Frontend/Alert%20X%20UI/Alert%20X/tailwind.config.ts)

For any issues, check your Node.js version and ensure all dependencies are installed.
